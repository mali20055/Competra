4.1 Fikst³r Otomatik Oluturma

Turnuva balatildiinda Flutter istemci tarafi fikst³r oluturur ve matches alt koleksiyonuna toplu yazar.

Lig Formati -- Round Robin

    Ę N oyuncu ińin N*(N-1)/2 toplam mań
    Ę Round robin algoritmasi: oyuncular daire olarak siralanir, her turda ńiftler eletirilir
    Ę N tekse bir 'bye' (bo) oyuncu eklenir
    Ę Her tur bir round deeri alir

Eleme Formati

    Ę Katilimcilar shuffle ile karitirilir, ńiftler oluturulur
    Ę Kazananlar bir sonraki tura geńer, matchId referansi ile balanir
    Ę Finali kazanan ampiyon olur
    Ę 2'nin kuvveti olmayan oyuncu sayisinda ilk turda 'bye' mańlari oluturulur

Grup + Eleme Formati

    Ę Oyuncular gruplara eit daitilir (tercihen 4'erli)
    Ę Her grup kendi ińinde round robin oynuyor
    Ę Her gruptan ilk 2 eleme aamasina geńiyor
    Ę Eleme aamasi standart eleme algoritmasiyla devam ediyor

ampiyonlar Ligi Formati

    Ę Lig fazi: her oyuncu belirli sayida rakiple oynuyor (t³m oyuncuyla deil)
    Ę Lig fazi sonunda ilk 8 (veya 16) eleme aamasina geńiyor
    Ę Eleme aamasi ńeyrek final  yari final  final eklinde ilerliyor

5. Firestore G³venlik Kurallari

T³m veri eriimi Firestore Security Rules ile kontrol edilir. Temel prensipler:
    Ę Kullanici sadece kendi profilini yazabilir
    Ę Turnuva verisini sadece katilimcilar okuyabilir
    Ę Mań sonucu giriini sadece ilgili oyuncular veya admin yapabilir
    Ę Feedback sadece yazilabilir, okunamaz (admin hariń)

5.1 Temel Kural Yapisi

   rules_version = '2';
   service cloud.firestore {

       match /databases/{database}/documents {

           // Kullanici profilleri
           match /users/{uid} {

              allow read: if request.auth != null;
              allow write: if request.auth.uid == uid;
           }

           // Kullanici adi rezervasyonlari
           match /usernames/{username} {

              allow read: if true;
              allow create: if request.auth != null

                                        && request.resource.data.uid == request.auth.uid;
              allow delete: if request.auth.uid == resource.data.uid;
           }

           // Turnuvalar
           match /tournaments/{tournamentId} {

              allow read: if isParticipant(tournamentId);
              allow create: if request.auth != null;
              allow update: if isAdmin(tournamentId);

              // Katilimcilar
              match /participants/{uid} {

                  allow read: if isParticipant(tournamentId);
                  allow create: if request.auth.uid == uid;
                  allow update: if isAdmin(tournamentId);
              }

              // Mańlar
              match /matches/{matchId} {

                  allow read: if isParticipant(tournamentId);
                  allow update: if isMatchPlayer(tournamentId, matchId)

                                           || isAdmin(tournamentId);
              }
           }

           // Geri bildirimler -- sadece yaz
           match /feedback/{id} {

              allow create: if request.auth != null;
              allow read: if false;
           }
       }
   }
